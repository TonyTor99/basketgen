# Basket Strategy Web

Браузерная программа на Python для генерации стратегий по баскетбольному сборщику.

Что умеет:
- загружает Excel-сборщик;
- находит доступные пары статистики `Д / Г`;
- генерирует стратегии вида `1 / 2 / X`;
- считает прибыль, ROI, винрейт, серии, просадку, train/test;
- показывает результат прямо в браузере;
- таблица результатов сортируется и фильтруется;
- можно открыть детали стратегии и посмотреть матчи, которые в неё попали.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate   # Linux / macOS
# .venv\Scripts\activate    # Windows

pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

После запуска откройте в браузере:

```text
http://127.0.0.1:5000
```

## Как использовать

1. На главной странице загрузите Excel-файл сборщика.
2. Выберите признаки, которые хотите использовать.
3. Укажите параметры генерации:
   - фильтр по боту;
   - время сигнала;
   - минимальное количество матчей;
   - top-k;
   - holdout days.
4. Нажмите «Сгенерировать стратегии».
5. Изучайте результаты в браузере.

## Рекомендации

- Для первого запуска берите 6-10 признаков.
- Если выбрать слишком много признаков, стратегий станет очень много и расчёт будет дольше.
- Для вашего файла логично начинать с времени сигнала `20`.
- В таблице результатов удобно сортировать по `ROI test`, `score`, `profit` и `matches`.

## Структура проекта

- `app.py` — запуск Flask-приложения.
- `basketgen/excel.py` — чтение Excel и подготовка данных.
- `basketgen/engine.py` — генерация стратегий и метрик.
- `basketgen/storage.py` — файловое хранение jobs.
- `templates/` — HTML-шаблоны.
- `static/` — CSS и JS.

## Важно

Программа не использует CDN и внешние библиотеки для интерфейса — всё работает локально.
